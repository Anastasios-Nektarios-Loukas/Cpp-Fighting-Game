# Cpp-Fighting-Game

## Description

A 2D fighting game developed in **C++** using the **SGG (Simple Game Graphics) library**.  
The project demonstrates core game development concepts such as the game loop, collision detection, and player combat mechanics.

It was developed as an **academic project for the C++ at the Athens University of Economics and Business (AUEB)**.

---

## Features

- Player vs Player combat  
- Hit / collision detection  
- Health system  
- Basic animations  
- Real-time gameplay  

---

## Technologies Used

- C++  
- SGG (Simple Game Graphics) Library  
- Visual Studio  

---

## How to Run

1. Open `FightingGame.sln` with Visual Studio  
2. Build the project (Ctrl + Shift + B)
3. Run the project (F5)
**OR**
Run the executable directly from the `bin/` folder.
